SAE 303 - VISUALISATION DE DONNEES : PRÉFÉRENCES SPOTIFY


AUTEUR : Timothé Roberjot
SUJET : Analyse des habitudes d'écoute selon l'âge

1. DESCRIPTION DU PROJET
Ce site montre une analyse de plus de 400 utilisateurs de Spotify.
L'objectif est de déterminer l'impact de l'âge sur les préférences de contenu sur l'application.

2. ROUTE DU SITE
Le site est un one page qui se visite en scrollant de haut en bas.
Les questions et graphiques s'enchaînent à la suite suivant la logique de la question principale.
Il est possible d'interagir avec les éléments (hover sur les sections, les cartes et les graphiques).

3. LANGUAGES / BIBLIOTHEQUE
- HTML5 / CSS3
- JavaScript
- Plotly.js

4. STRUCTURE DU DOSSIER
- index.html
- style.css
- script.js
- readme.txt

5. SOURCES ET IA
- Données : Spotify_users_data.csv
- Aide externe : Utilisation ponctuelle de l'IA Gemini pour le CSS (notamment sur le positionnement des blocs d'analyse et la cohérence visuelle)
  et le JS (telle que pour la compréhension des erreurs de configuration dans Plotly et des ajustements simples de fonctions)
