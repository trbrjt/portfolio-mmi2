Aperçus image des livrables PDF / print
=======================================

Déposez ici les exports image (PNG ou JPG) des pages clés. Les noms de fichiers
attendus par index.html sont EXACTEMENT :

  marsup-1.png        (plaquette MARSUP — 1 page clé ; option : marsup-2.png)
  pop-smoke-1.png     (dossier Pop Smoke — planche 1)
  pop-smoke-2.png     (dossier Pop Smoke — planche 2)
  pop-smoke-3.png     (dossier Pop Smoke — planche 3)
  pop-smoke-4.png     (dossier Pop Smoke — planche 4)

Tant qu'un fichier est absent, l'emplacement affiche un placeholder discret et le
lien vers le PDF complet reste fonctionnel — aucune image cassée n'apparaît.

Conseils d'export :
- Largeur ~1200–1600px, qualité élevée, fond inclus.
- Pour Pop Smoke, choisir des planches qui montrent la DA noir/or/argent.
- Si vous changez les noms, adaptez les <img src> correspondants dans src/index.html.
