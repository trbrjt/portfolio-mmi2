# Portfolio de compétences — BUT MMI2

**Timothée Roberjot** · Parcours : Stratégie de communication numérique et design d'expérience (SCN2D)
Année universitaire 2024-2025 · IUT MMI

## Objectif

Démontrer l'acquisition des compétences et apprentissages critiques du BUT MMI à travers l'analyse de missions de stage/alternance et de SAEs, structurée selon la méthode STAR.

## Lancer le projet

Aucune dépendance, aucun serveur requis.

```bash
# Ouvrir directement dans un navigateur :
start src/index.html       # Windows
open src/index.html        # macOS
```

Ou double-clic sur `src/index.html` dans l'explorateur de fichiers.

## Structure du projet

```
src/
├── index.html              # Page d'entrée principale
├── styles/main.css         # Design system (variables, composants)
├── scripts/main.js         # Navigation et interactivité minimale
└── content/

assets/
├── fonts/                  # Polices locales
└── images/                 # Visuels de projets
contexte.md                 # Consignes et critères pédagogiques officiels

CLAUDE.md                   # Instructions pour Claude Code
```

## Approche méthodologique et outillage

Dans une démarche d'optimisation et d'alignement avec les pratiques professionnelles du secteur de la commtech et du développement, ce portfolio a été réalisé en flux de production augmenté (Vibe Coding).

- **Conception, stratégie et fond :** 100% humain. L'analyse des missions, l'auto-évaluation réflexive, le ciblage des apprentissages critiques (AC) et la structure narrative (variantes de la méthode STAR) sont le fruit direct de mes deux années d'expérience en BUT MMI et de mon stage.
- **Production technique et relecture :** Utilisation ciblée d'assistants IA (Claude Code) comme outils d'intégration HTML/CSS vanilla, de refactoring de scripts et de correction syntaxique.

Cette approche m'a permis de me concentrer sur la pertinence stratégique de mon auto-évaluation tout en garantissant une rigueur technique et une intégrité du code optimales pour le livrable final.
